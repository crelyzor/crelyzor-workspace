## Next Steps

Manual actions required that the pipeline couldn't do automatically:

1. **Schema migration needed**: `MeetingNote.author` should be a proper FK to `User.id`. Currently a bare string. Requires `prisma migrate` + data migration script.

2. **exportContacts async queue**: The synchronous card contact export in `cardService.ts` should be moved to a Bull queue job to avoid blocking the event loop on large exports.

3. **Pagination for /recordings and /cards list endpoints**: Both endpoints return unbounded lists. Needs coordinated backend + frontend change (add pagination params + update React Query hooks).

4. **TypeScript `any` cleanup**: Several files still use `any` types in non-security-critical paths. These should be addressed in a dedicated type-safety pass.

5. **RECALL_WEBHOOK_SECRET HMAC verification**: The startup warning was added, but if this env var is missing in production, Recall.ai webhooks are processed without signature verification. Ensure the env var is set in all production environments.
